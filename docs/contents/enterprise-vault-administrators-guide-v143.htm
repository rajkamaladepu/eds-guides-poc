Maximum concurrent indexing tasks (Site Properties Indexing setting)

Description
Specifies the maximum number of concurrent indexing tasks.
Note:
Do not change this advanced setting unless your technical support provider advises you to do so.
Supported values
  • An integer in the range 1 to 25. The default is 20.
