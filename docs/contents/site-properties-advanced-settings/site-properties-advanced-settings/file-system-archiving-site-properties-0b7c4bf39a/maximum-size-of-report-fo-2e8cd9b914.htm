Maximum size of report for rollover (site properties File System Archiving setting)

Description
Specifies the rollover size of the report (in MB). If the size of the report exceeds the specified limit, a new report file is created.
Supported values
An integer in the range 0 to 1024. The default value is 100. Set the value to 0 to disable rollover.
