Age-based retention of report (site properties File System Archiving setting)

Description
Controls whether age-based retention is enabled for the File System Archiving reports. Note that this setting overrides Number of reports to keep on the Reports tab in the File System Archiving task properties.
Supported values
  • No. Reports are retained based on the setting on the Reports tab in the task properties.
  • Yes. Files are retained for the number of days specified in the Days to keep report setting.
