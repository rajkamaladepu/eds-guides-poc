NetApp C-Mode server certificate verification (site properties File System Archiving setting)

Description
Lets you specify whether the ONTAPI Management API should verify the server certificate that is issued to the NetApp C-Mode server. To verify the server certificate, make sure that the certificate of the certificate issuing authority is in the cacert.pem file in the Enterprise Vault installation folder.
Supported values
Verify certificate.
Verify certificate, but ignore the hostname mismatch. Set this value if you want the ONTAPI Management API to verify the certificate, but ignore any mismatch that is found between the host name specified in the certificate and the host name of the management LIF.
Do not verify certificate (default).
