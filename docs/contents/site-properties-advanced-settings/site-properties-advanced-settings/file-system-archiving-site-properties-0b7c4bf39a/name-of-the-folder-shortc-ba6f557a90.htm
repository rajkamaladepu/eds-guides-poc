Name of the folder shortcut file (site properties File System Archiving setting)

Description
Lets you change the name of the folder shortcut file (.url), which contains a hypertext link to the archived folder. The file name would be .url. The default name is [View Archived Files].url. Note that if you have customized the folder shortcut name, you may see multiple folder shortcut files. The File System Archiving task does not delete old .url files.
Supported values
Any string value that contains the characters that the file system supports. Note that the folder shortcut file name cannot contain any of the following characters:
< (less than), > (greater than), : (colon), " (double quote), / (forward slash), \ (backslash), | (vertical bar or pipe), ? (question mark), * (asterisk), and the characters whose integer representations are in the range from 1 through 31.
