Location of the report (site properties File System Archiving setting)

Description
Specifies the path where the report is stored. The path should be local to each server in the site that hosts the File System Archiving task.
Supported values
A text value specifying a path. The default path is the Reports\FSA subfolder of the Enterprise Vault installation folder.
