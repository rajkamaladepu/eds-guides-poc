Days to keep report (site properties File System Archiving setting)

Description
Specifies the number of days to keep the File System Archiving reports on the server. The File System Archiving task deletes the reports that are older than the specified number of days. You must set the Age-based retention of report setting to Yes to use this setting.
The purging of the report files that are eligible for deletion happens according to the schedule that you define for the File System Archiving task.
Supported values
An integer in the range 1 to 999. The default is 30.
