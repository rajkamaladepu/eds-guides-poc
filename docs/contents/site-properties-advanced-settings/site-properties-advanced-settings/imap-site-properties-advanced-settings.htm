IMAP (site properties advanced settings)

The IMAP settings let you control advanced IMAP behavior.
The IMAP settings are:
  • Folder limit (site properties IMAP setting)
  • Caching duration (site properties IMAP setting)
  • Caching duration when disk space is low (site properties IMAP setting)
  • Send Email Timeout (site properties IMAP setting)
