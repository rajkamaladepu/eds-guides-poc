Folder limit (site properties IMAP setting)

Folder limit (site properties IMAP setting)

Description
Limits the number of archived items that Enterprise Vault returns to IMAP clients from each archive folder. Enterprise Vault returns the most recently archived items, up to the limit you set.
Supported values
  • An integer that sets the folder limit. The default is 10000.
