List of internal SMTP domains (Site Properties SMTP setting)

Description
Specifies a list of SMTP domains that are to be considered internal to the company. For example, "ourcompany.com;ourcompany.co.ie;ourcompany.co.uk". These domains are supplemental to those that Enterprise Vault detects from the SMTP addresses of any Exchange system mailboxes in the site. Alternatively, you can use an asterisk (*) to specify that all SMTP domains are internal.
This setting applies to both SMTP Archiving and Skype for Business Archiving.
Supported values
  • A list of domains separated by semi-colons.
