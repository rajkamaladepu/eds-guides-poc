Search custom header Recipient (Site Properties SMTP setting)

Description
This setting configures the SMTP Archiving task to search all recipient under the Recipient field in each message.
This setting applies to SMTP Archiving only.
To optimize performance for SMTP Archiving, ensure that this advanced site setting is set to No.
Supported values
  • No \- By default, the value is set to No, so that SMTP target addresses in the Recipient field are not matched.
  • Yes \- If the value is set to Yes, SMTP target addresses in Recipient field are matched.
