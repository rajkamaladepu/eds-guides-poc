Delete messages without recipients or a matching target (Site Properties SMTP setting)

Description
Controls what action to take when message files in the holding folder do not have recipients or a matching target.
This setting applies to both SMTP Archiving and Skype for Business Archiving.
Supported values
  • No. Do not delete. Messages that do not contain a matching target address that is enabled for archiving are moved to the folder NoMatchingTarget , which is in the holding folder. In the NoMatchingTarget folder, subfolders for day, hour, and minute are created as needed. The message file is placed in the appropriate minute folder.
  • Yes (default). Delete messages without recipients or a matching target
