File types for Postscript conversion (site properties Content Conversion setting)

Description
The list of file types to convert using Enterprise Vault Postscript conversion.
Note that this setting does not override the setting, File types excluded from conversion. For example, if the file type .xyz is included in both setting values, then instances of the file type xyz are not converted.
Supported values
You can list file types in the format:
.filetype
By default, the following file types are converted using Enterprise Vault Postscript conversion:
.PS.EPS
