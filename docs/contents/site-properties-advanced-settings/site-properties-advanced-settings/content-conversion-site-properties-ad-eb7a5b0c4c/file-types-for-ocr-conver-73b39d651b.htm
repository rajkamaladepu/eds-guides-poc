File types for OCR conversion of embedded images (site properties Content Conversion setting)

Description
The list of file types to which Optical Character Recognition (OCR) conversion of embedded images is applied, when converting document items to HTML.
Supported values
You can list file types in the format:
.filetype
Setting the value to an empty string effectively disables OCR conversion of embedded images.
Specifying the wildcard value "*" includes all supported file types. (Default value)
See the description of the OCR Conversion of embedded images Content Conversion setting for information about the supported document items on which you can apply OCR conversion of embedded images.
