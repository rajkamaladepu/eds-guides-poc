File types for IFilter conversion (site properties Content Conversion setting)

Description
The list of file types to convert using Windows IFilter conversion.
The relevant 64-bit IFilter for each configured file type must be installed on all the Enterprise Vault servers in the site that host a Storage Service.
Note that this setting does not override the setting, File types excluded from conversion. For example, if the file type .xyz is included in both setting values, then instances of the file type xyz are not converted.
Supported values
You can list file types in the format:
.filetype
