Log fallback to text events (site properties Content Conversion setting)

Description
Controls whether to record in the event log successful conversions to text after a failed conversion to HTML. These entries are recorded in the Enterprise Vault Converters event log.
Note that the value of this setting does not override the Log conversion failure events setting.
Supported values
  • On (default). Successful fallback conversions to text are recorded in the event log.
  • Off. Successful fallback conversions to text are not recorded in the event log.
Legacy registry setting
  • Storage\FallbackConversionEvents
