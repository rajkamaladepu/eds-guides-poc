Conversion timeout for archive file types (site properties Content Conversion setting)

Description
The conversion timeout in minutes for archive file types, for example, ZIP files.
Supported values
The value is specified as an integer.
The default value is 10.
Legacy registry setting
  • Enterprise Vault\ConversionTimeoutArchiveFiles
