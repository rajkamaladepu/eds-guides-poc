OCR conversion of scanned pages (site properties Content Conversion setting)

Description
Controls whether to apply Optical Character Recognition (OCR) conversion to scanned pages, when converting PDF items to HTML.
The setting is only effective when JPG or PNG file types are configured for OCR or Windows IFilter conversion.
Supported values
  • On. OCR conversion of scanned pages is enabled.
  • Off (default). OCR conversion of scanned pages is disabled.
