Include hidden spreadsheet data (site properties Content Conversion setting)

Description
Controls whether the contents of hidden cells, columns, and sheets are included when converting spreadsheet items to HTML.
Supported values
  • On (default). Hidden data is included in the conversion.
  • Off. Hidden data is not included in the conversion.
Legacy registry setting
  • Enterprise Vault\ConversionIncludeHiddenSpreadsheetData
