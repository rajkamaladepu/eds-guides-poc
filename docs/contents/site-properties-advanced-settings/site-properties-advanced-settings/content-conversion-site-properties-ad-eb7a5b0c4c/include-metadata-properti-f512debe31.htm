Include metadata properties (site properties Content Conversion setting)

Description
When converting items to HTML, specifies whether to include the metadata properties for file types that contain embedded metadata.
The set of metadata properties depends on whether the file type content contains embedded properties, and the content converter that is configured for the file type. For example, the metadata properties for a photo file could include camera details. For PDF and Microsoft Office documents, the metadata properties could include author, subject, and created and last modified dates.
Supported values
  • On (default). Include the metadata properties for file types that contain embedded metadata.
  • Off. Do not include the metadata properties for these file types.
