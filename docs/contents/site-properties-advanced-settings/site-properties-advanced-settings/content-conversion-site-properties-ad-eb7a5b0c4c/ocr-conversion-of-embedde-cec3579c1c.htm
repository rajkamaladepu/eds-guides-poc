OCR Conversion of embedded images (site properties Content Conversion setting)

Description
Controls Optical Character Recognition (OCR) conversion of embedded images when converting document items to HTML. Document items include PDF and Microsoft Office documents. Embedded images in PDF documents, include pages that have been scanned in from printed documents. In Office documents, embedded images include charts, diagrams, and so on.
The setting is only effective when JPG or PNG file types are configured for OCR or Windows IFilter conversion.
Note:
Enabling OCR conversion of embedded images may impact archiving performance.
Supported values
  • On. OCR conversion of embedded images is enabled.
  • Off (default). OCR conversion of embedded images is disabled.
