Include hidden text (site properties Content Conversion setting)

Include hidden text (site properties Content Conversion setting)

Description
Controls whether hidden text is included when converting document items to HTML.
PDF and Microsoft Word are example file types that can contain hidden text.
Supported values
  • On (default). Hidden text is included in the conversion.
  • Off. Hidden text is not included in the conversion.
Legacy registry setting
  • Enterprise Vault\ConversionIncludeHiddenText
