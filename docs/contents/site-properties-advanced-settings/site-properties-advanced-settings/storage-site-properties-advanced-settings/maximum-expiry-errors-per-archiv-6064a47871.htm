Maximum expiry errors per archive (Site Properties Storage setting)

Description
Specifies the maximum number of errors per archive that may occur while processing items for automatic expiry. When this limit is exceeded, Enterprise Vault abandons expiry for the archive. However, it tries again when automatic expiry next runs.
Supported values
  • An integer in the range 0 through 100000. The default is 20.
