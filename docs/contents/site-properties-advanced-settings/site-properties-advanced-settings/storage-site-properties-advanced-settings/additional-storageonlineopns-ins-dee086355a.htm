Additional StorageOnlineOpns instances (Site Properties Storage setting)

On an Enterprise Vault storage server there can never be less than one StorageOnlineOpns process. This setting controls the number of extra processes StorageOnlineOpns to run. The default is for four extra processes, making a total of five StorageOnlineOpns processes.
Note:
Do not change this advanced setting unless your technical support provider advises you to do so.
