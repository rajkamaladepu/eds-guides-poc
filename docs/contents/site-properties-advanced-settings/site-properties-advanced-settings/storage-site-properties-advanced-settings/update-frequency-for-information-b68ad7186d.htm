Update frequency for Information Classifier configuration file (Site Properties Storage setting)

Description
Note:
This setting is for use with the Veritas Information Classifier in Enterprise Vault 12.2 and later. It has no effect on classification using the Microsoft File Classification Infrastructure (FCI).
Specifies the frequency in milliseconds with which Enterprise Vault should update the configuration settings for all instances of the Veritas Information Classifier engine in the site. These settings include the path to the network folder in which to keep classification policy information, and the required port and protocol for accessing the Veritas Information Classifier.
Enterprise Vault stores the settings on a per-engine basis in configuration (YAML) files. Each time a request is made to the Veritas Information Classifier engine, or you start or restart it, Enterprise Vault automatically updates the configuration files. Enterprise Vault then updates the files with the frequency that you specify here.
You may find that increasing the value of this setting reduces the network traffic on your servers and marginally improves classification performance. Otherwise, however, we recommend that you do not change this setting.
Supported values
An integer in the range 100 through 100000. The default is 5000 (5 seconds).
