Enforce sentiment analysis (site properties Classification setting)

  • Description: Controls whether to enforce sentiment analysis during classification. Sentiment analysis is enforced irrespective of any policy for which sentiment condition is enabled or not. Sentiment analysis is applicable only to Veritas Information Classifier.
  • Supported values:
  • OnSentiment analysis is enforced during classification.
  • Off (default). Sentiment analysis is not enforced during classification.
