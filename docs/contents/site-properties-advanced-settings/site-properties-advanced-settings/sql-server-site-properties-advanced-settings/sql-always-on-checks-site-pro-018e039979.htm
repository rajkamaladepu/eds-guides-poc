SQL Always On checks (site properties SQL Server setting)

Description
Use to enable and disable checks for availability groups and failover cluster instances on a SQL server when you create Enterprise Vault databases. Set to Enabled to show informational messages, and Disabled to hide them.
Supported values
  • Enabled (default).
  • Disabled.
