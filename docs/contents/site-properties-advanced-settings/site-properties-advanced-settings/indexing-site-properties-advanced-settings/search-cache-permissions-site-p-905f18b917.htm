Search Cache Permissions (Site Properties Indexing setting)

Search Cache Permissions (Site Properties Indexing setting)

Description
Specifies whether or not user permissions are cached in the Index Query Server.
Note:
Do not change this advanced setting unless your technical support provider advises you to do so.
Supported values
  • On (default). User permissions are cached in the Index Query Server.
  • Off. User permissions are not cached in the Index Query Server.
