Search Max Attempts When Engine Busy (Site Properties Indexing setting)

Description
The maximum number of attempts for searches that fail due to the search engine being temporarily too busy to handle the search.
Note:
Do not change this advanced setting unless your technical support provider advises you to do so.
Supported values
  • An integer specifying a number of search attempts. The default is 3.
