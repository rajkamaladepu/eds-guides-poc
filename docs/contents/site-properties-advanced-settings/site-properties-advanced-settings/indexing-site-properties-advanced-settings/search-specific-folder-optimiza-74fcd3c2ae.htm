Search Specific Folder Optimization (Site Properties Indexing setting)

Description
Specifies whether or not to apply a performance optimization to searches that target a small number of folders. The Search Max Folders For Specific Folder Optimization setting determines the limit for what is considered small.
See Search Max Folders For Specific Folder Optimization (Site Properties Indexing setting).
Note:
Do not change this setting unless your Technical Support provider advises you to do so.
Supported values
  • On (default). Applies the performance optimization.
  • Off. Performance optimization is not applied.
