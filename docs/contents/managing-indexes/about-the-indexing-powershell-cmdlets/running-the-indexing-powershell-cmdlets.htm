Running the indexing PowerShell cmdlets

To run one of the indexing PowerShell cmdlets
  1. Start the Enterprise Vault Management Shell.
PowerShell opens and loads the Enterprise Vault snap-in. The cmdlets are now available in the shell.
  1. At the PowerShell command prompt, type the required command.
Help is available for all the cmdlets. For example, the following command shows the detailed Help for the Get-IndexServerForIndexLocation cmdlet:
Get-Help Get-IndexServerForIndexLocation -detailed
