Using Get-EVElasticsearchIndex

Use the following syntax when you run Get-EVElasticsearchIndex :
Get-EVElasticsearchIndex -ArchiveId
Parameter
Description
-ArchiveId (required)
The ID for the archive for which you want to retrieve the details of the indices.
-IndexState
The state (Open/Close) of the index with which you want to filter the results.
  • Get-EVElasticsearchIndex -ArchiveId "116C02E87F248354DA9E9FECEDA650C721110000VJ-SQL.QA.COM"
  • Get-EVArchive | Get-EVElasticsearchIndex
  • Get-EVElasticsearchIndex -ArchiveId "116C02E87F248354DA9E9FECEDA650C721110000VJ-SQL.QA.COM" -IndexState "Open"
  • Get-EVArchive | Get-EVElasticsearchIndex -IndexState "Close"
