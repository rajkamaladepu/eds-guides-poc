Managing indexing exclusions

For each Enterprise Vault site, you can manage indexing exclusions in the Administration Console.
To manage indexing exclusions
  1. In the Enterprise Vault Administration Console, open Site Properties and click the Indexing tab.
  2. Click Exclusions.
In the Indexing Exclusions dialog box, you can add new indexing exclusions, and edit, rename and remove existing indexing exclusions.
