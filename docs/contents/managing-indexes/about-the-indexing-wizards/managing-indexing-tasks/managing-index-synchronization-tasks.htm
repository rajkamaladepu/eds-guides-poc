Managing index synchronization tasks

Synchronization tasks which do not complete successfully with the status No errors, can have the following statuses:
  • Warnings. Synchronization subtasks can complete with Warnings. In the Monitor Indexing Tasks page, open the subtask's report file to see the warnings.
  • Failed. The synchronization subtask has failed. In the Monitor Indexing Tasks page, open the subtask's report file to see details of the failure. If you can ascertain and rectify the cause of the failure, use the Retry action in the Monitor Indexing Tasks page to retry the subtask.
