Assigning the Extension Content Provider Application role

The account under which an extension runs must be assigned to the Extension Content Provider Application role.
See Roles-based administration.
The Extension Content Provider Application role lets the application to run as an Enterprise Vault extension, to create, read, and update extension entries in the directory, and to archive items.
This role does not allow full update permission on all extension properties. For example an extension cannot enable or disable itself, nor can it modify or override its own schedule.
