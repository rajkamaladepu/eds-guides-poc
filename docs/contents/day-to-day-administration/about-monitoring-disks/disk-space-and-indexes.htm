Disk space and indexes

If you are running out of disk space for indexes, you must provision more space or move indexes to new location with sufficient amount of space. For more information, see How to move the Index location to a different location on the same server.
Enterprise Vault does not create new indexes in a closed location or update existing indexes that are in a closed location. Enterprise Vault automatically creates new indexes in an open location, as required.
