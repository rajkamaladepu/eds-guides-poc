Checking disk space for vault stores

You must ensure that there is enough disk space for the vault store databases. When your Enterprise Vault installation has reached its normal running state, you can estimate the rate at which disk space is consumed and so be able to plan accordingly.
If a disk is running out of space, you can use SQL Enterprise Manager to create a new SQL device on another disk and then expand the database on to the new device.
