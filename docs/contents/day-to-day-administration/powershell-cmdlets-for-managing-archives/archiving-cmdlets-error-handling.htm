Archiving cmdlets error handling

The archiving cmdlets write the error information directly on the PowerShell host.
Table: Archiving cmdlet errors describes the standard set of errors that the cmdlets return.
Table: Archiving cmdlet errors
ErrorID
Reason for failure
Action
ArchiveAlreadyMarkedForDeletion
The specified archive is already marked for deletion.
No action required.
ArchiveAssociatedWithTargets
You cannot delete this archive because it is associated with one or more targets.
Configure the listed targets to use a different archive and retry the operation. If archive is associated with a SharePoint Site Collection, delete the Site Collection from Enterprise Vault and retry the operation.
ArchiveInUse
The specified archive is in use. You can only delete archives that have a status of 'Available' or 'Closed'.
Wait till the archive is not in use and retry the operation.
ArchiveNotFoundInDirectory
The specified archive is not found in the Enterprise Vault directory.
Verify whether the archive is already deleted from Enterprise Vault. Make sure that the specified archive ID corresponds to an existing archive.
ArchiveProtectedFromDeletion
The specified archive is protected from deletion.
In the Archive Properties dialog box, on the Advanced tab, select Allow deletion of archived items and of this archive .
ArchiveUnderInProgressMoveOperation
The specified archive cannot be deleted because it is being moved.
Check the Move Archive Status page in the Administration Console. Retry the deletion after the Move Archive operation has completed successfully.
AssociatedVaultStoreInBackupMode
The vault store that is associated with the specified archive is in backup mode.
Use Clear-VaultStoreBackupMode to clear the backup mode from the vault store.
EnterpriseVaultDirectoryServerNotReachable
The Enterprise Vault directory server is not reachable.
Make sure that the archive ID provided as input belongs to an existing Enterprise Vault directory.
EnterpriseVaultDirectoryServiceNotAvailable
The Enterprise Vault Directory service is not running.
Start the Enterprise Vault Directory Service and retry the deletion.
EnterpriseVaultStorageServerNotReachable
The Enterprise Vault storage server is not reachable.
Check whether the Enterprise Vault storage server is reachable, and the user running the cmdlet has appropriate permissions on the server.
EnterpriseVaultStorageServiceNotAvailable
The Enterprise Vault Storage service is not running.
Start the Enterprise Vault Storage service and retry the deletion.
FailedToDetermineCustomArchiveType or UnexpectedError
An unexpected error has occurred.
Check the event logs and DTrace logs for more information.
InvalidFormat
The archive ID is not in a valid format.
Make sure that you specify the archive ID in the correct format.
NullEmptyWhitespaceNotAllowed
The archive ID contains white spaces or the input that is provided is Null or Empty.
Make sure that you specify a valid archive ID.
UnauthorizedAccess
You are not authorized to perform this operation.
Ensure that you have the appropriate permissions to delete the specified archive.
