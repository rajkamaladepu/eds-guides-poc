Use the default ANSI codepage (ACP codepage)

Enterprise Vault has a list of codepages in the file CodePages.txt , which is installed in the Enterprise Vault program folder (for example C:\Program Files (x86)\Enterprise Vault).
You can force Enterprise Vault to use the default ANSI codepage (ACP codepage) by deleting the appropriate codepage entry from the file. If you find that the ACP codepage does not give you good conversions, replace the entry in Codepages.txt and install the missing codepage.
To delete a codepage entry
  1. Edit CodePages.txt on the Exchange Journaling task computer to remove the entry for the codepage that caused the problem.
  2. To prevent a similar problem happening on Exchange Mailbox task computers, make the same change to CodePages.txt on each Exchange Mailbox task computer.
  3. If you use a building blocks configuration, repeat the change on every Enterprise Vault server in your site.
