Selecting archives

Use this page to select the archives that you want to move, and to select destination archives. Only eligible archives are shown.
Move Archive supports move operations for the following archive types:
  • Domino mailbox and journal archives
  • Exchange mailbox and journal archives
Move Archive does not support the following archive types:
  • File System Archiving (FSA) archives
  • SharePoint archives
  • Shared archives
  • Exchange public folder archives
  • Internet Mail archives
  • SMTP archives
Move Archive also prevents moves in the following circumstances:
  • The source archive exceeds its archive usage limit and the destination is a new archive.
  • The source archive is closed.
  • The source archive contains items that Discovery Accelerator has placed on legal hold.
  • The destination archive exceeds its archive usage limit.
When an archive exceeds its usage limit, you should increase the limit on the Archive Properties: Archive Usage Limit tab in the Administration Console.
To add archives
  1. Click Add.
  2. Use the Select Source dialog box to add the archives you want to move.
When you move an archive that is currently in use by an archiving-enabled user, you can only move it to a new archive. In this case,
  1. Select the source archive for which you want to select a destination archive.
  2. Click Move To and click Select Destination.
  3. Use the Select Destination dialog box to select a destination archive.
