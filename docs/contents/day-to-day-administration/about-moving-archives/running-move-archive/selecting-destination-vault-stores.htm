Selecting destination vault stores

If you have chosen to move any archives to new destination archives, this page lets you choose a destination vault store for each. Each destination archive must be in a different vault store from the associated source archive.
To select destination vault stores
  1. Select one or more source archives.
  2. Click Select Destination.
  3. Use the Select Vault Store dialog box to select a destination vault store for the selected archives.
