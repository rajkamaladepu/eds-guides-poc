Displaying the status page

On this page, select the Show the 'Move Archive Status' dialog option if you want to monitor the progress of the archive moves.
The first time you move an archive associated with a particular storage server, Enterprise Vault creates a Move Archive task on that server. The task is configured to start automatically each time the task controller service starts. If you want to start the task straight away, select the Start the new tasks now option. Enterprise Vault shows this option only once for each storage server.
