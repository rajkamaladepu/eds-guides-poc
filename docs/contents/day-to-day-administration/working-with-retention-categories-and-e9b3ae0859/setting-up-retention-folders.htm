Setting up retention folders

Enterprise Vault provides several methods for setting up retention folders:
  • In the Administration Console, you can set up retention folders when you run the New Retention Plan wizard. Then, after you have created the retention plan, you can add new retention folders to it and modify the existing retention folders by editing the properties of the retention plan.
  • You can define the attributes of the retention folders in an XML file and then use PowerShell cmdlets to associate this file with one or more retention plans.
