Monitoring MSMQ queues

It is important that you monitor MSMQ queues so that you can quickly spot any problems that may occur.
To monitor the performance of the queues:
  • Use the Windows Performance Monitor
You may find it useful to have the Windows Performance Monitor running continuously, showing the number of messages on all the queues.
You will quickly become used to the normal behavior of the queues and will notice excessive backlogs. Investigate the cause of any such backlogs promptly.
