Monitoring database maintenance

The Monitoring database contains the data that is gathered by the Enterprise Vault Monitoring agents. The data is used by Enterprise Vault Operations Manager, and in some of the reports generated by Enterprise Vault Reporting.
Ensure that there is adequate space for the database to grow as data is added.
The initial size of the Monitoring database is as follows:
Data device
100 MB
Transaction log
80 MB
Total
180 MB
The Monitoring database components are named as follows:
Database name
EnterpriseVaultMonitoring
Data device name
EnterpriseVaultMonitoring_Data
Log device name
EnterpriseVaultMonitoring_Log
