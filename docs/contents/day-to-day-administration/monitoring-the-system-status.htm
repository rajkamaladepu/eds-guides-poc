Monitoring the system status

Enterprise Vault automatically runs checks to monitor the health of the Enterprise Vault system.
Enterprise Vault displays the results of the checks in the Status section of the Administration Console. Check the status results every day and fix any problems that are shown.
To customize the checks for your environment you can do the following:
  1. Choose which checks to run.
  2. Modify the levels at which the checks report errors.
  3. Specify when to run each check.
See Monitoring in Site Properties.
