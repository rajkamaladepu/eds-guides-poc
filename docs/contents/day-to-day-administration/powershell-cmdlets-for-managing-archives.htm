PowerShell cmdlets for managing archives

Table: PowerShell cmdlets for managing archives lists the PowerShell cmdlets that the Enterprise Vault Management Shell provides for managing archives. See the PowerShell Cmdlets guide for more information on them.
Table: PowerShell cmdlets for managing archives
Cmdlet
Description
Get-EVArchive
Returns a list of some or all of the archives in the Enterprise Vault site.
Remove-EVArchive
Removes the specified archive from the Enterprise Vault site.
Set-EVArchive
Updates the specified settings of an existing archive.
