PowerShell cmdlets for managing archive permissions

Table: PowerShell cmdlets for managing archive permissions lists the PowerShell cmdlets that the Enterprise Vault Management Shell provides for managing archive permissions. For the most part, these cmdlets duplicate facilities that are available in the Administration Console. The same information is also available in the PowerShell Cmdlets guide.
Table: PowerShell cmdlets for managing archive permissions
Cmdlet
Description
Get-EVArchivePermission
Returns a list of all the users and groups who have access permissions on the specified archive.
See Get-EVArchivePermission.
Remove-EVArchivePermission
Removes the access permissions that are granted or denied to a given user or group on the specified archive.
See Remove-EVArchivePermission.
Set-EVArchivePermission
Grants or denies access permissions to a given user or group on the specified archive.
See Set-EVArchivePermission.
