Retention category selection (Classification Policy setting)

Retention category selection (Classification Policy setting)

Description
Specifies the retention category to use when classification tags match multiple retention categories. Enterprise Vault uses the retention category that retains the item for the longest or shortest duration, depending on this setting value.
Supported values
  • Longest (default)
  • Shortest
Legacy name
