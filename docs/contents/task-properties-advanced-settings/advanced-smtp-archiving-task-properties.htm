Advanced SMTP Archiving task properties

The advanced SMTP Archiving task properties control advanced aspects of SMTP Archiving tasks.
The advanced SMTP Archiving task properties are as follows:
  • Checkpoint interval for the SMTP Archiving task
  • Frequency to update SMTP Archiving task Summary Report
  • Maximum occurrences of unchanged checkpoints
