Checkpoint interval for the SMTP Archiving task

Description
The time between the SMTP Archiving task checkpoints, in minutes.
The SMTP Archiving task performs checkpointing at regular intervals. If the task is stopped for any reason, the checkpoint information enables it to resume from the most recent checkpoint position.
This setting controls how frequently the task saves checkpoint information.
Supported values
  • An integer number of minutes. The default is 720 (minutes).
