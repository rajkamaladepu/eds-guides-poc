Maximum occurrences of unchanged checkpoints

Description
The SMTP Archiving task performs checkpointing at regular intervals. If the checkpoint information remains unchanged then it is likely that the there is a problem with the task. This setting specifies the maximum number of times an unchanged checkpoint is allowed before the SMTP Archiving task logs a warning event.
The SMTP Archiving task also generates error log files in the following folder:
_Enterprise_Vault_installation_folder_ \Reports\SMTP\_SMTP_task_name_
Supported values
  • An integer number. The default is 6.
