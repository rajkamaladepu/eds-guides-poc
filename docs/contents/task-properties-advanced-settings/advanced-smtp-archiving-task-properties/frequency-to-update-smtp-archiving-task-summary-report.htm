Frequency to update SMTP Archiving task Summary Report

Description
The time interval between updates of the SMTP Archiving task summary information in the task report file (minutes).
The archiving task generates SMTP archiving summary reports and task error log files in the following folder:
_Enterprise_Vault_installation_folder_ \Reports\SMTP\_SMTP_task_name_
The task also creates Skype for Business archiving summary reports in the following subfolder:
_Enterprise_Vault_installation_folder_ \Reports\SMTP\_SMTP_task_name_ \Skype for Business
Supported values
  • An integer value. The default is 60 (minutes).
