Thread pool maximum size (Computer Properties IMAP setting)

Description
Use this property to configure the maximum number of CPU threads that are available to service IMAP requests.
Note that the actual number of threads can be further constrained if the Thread pool algorithm is set to Hardware, and the computer's physical thread pool size is lower than the limit you set.
Note:
Do not change this advanced setting unless your technical support provider advises you to do so.
Supported values
  • An integer in the range 20 to 400. The default is 100.
