Generate transcripts (Computer Properties IMAP setting)

Description
Use this property to generate transcripts for IMAP sessions on this Enterprise Vault server, when DTrace is running against IMAP. This can be useful to troubleshoot users' connection problems. Enterprise Vault writes transcripts to the Vault Service account's TEMP directory, in files whose extension is .X
Enterprise Vault deletes transcript files when you stop the IMAP server.
Note:
IMAP transcripts contain sensitive information from users' email.
Supported values
  • No (default).
  • Yes.
