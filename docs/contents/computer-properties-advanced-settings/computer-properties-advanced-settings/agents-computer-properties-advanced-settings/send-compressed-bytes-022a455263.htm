Send compressed bytes (Computer Properties Agents setting)

Send compressed bytes (Computer Properties Agents setting)

Description
Controls whether Enterprise Vault agents that archive from Exchange Server compress data before sending it to the storage queue. The compression can reduce network traffic on a WAN. On fast networks the performance overhead of the compression is more significant than the network traffic gain.
This option is not recommended for fast networks.
Supported values
  • Yes.
  • No (default).
