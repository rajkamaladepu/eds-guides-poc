Maximum indexing application pool start time (Computer Properties Indexing setting)

Description
Maximum time to wait for the Indexing application pool to start, in milliseconds.
Note:
Do not change this advanced setting unless your technical support provider advises you to do so.
Supported values
  • An integer specifying a number of milliseconds. The default is 800.
