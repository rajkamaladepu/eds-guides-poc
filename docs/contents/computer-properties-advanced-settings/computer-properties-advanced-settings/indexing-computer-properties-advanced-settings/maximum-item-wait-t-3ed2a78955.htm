Maximum item wait time (Computer Properties Indexing setting)

Description
The maximum time the Indexing service waits for an item to be fetched from the Storage service (minutes).
Note:
Do not change this advanced setting unless your technical support provider advises you to do so.
Supported values
  • An integer specifying a number of minutes. The default is 30.
