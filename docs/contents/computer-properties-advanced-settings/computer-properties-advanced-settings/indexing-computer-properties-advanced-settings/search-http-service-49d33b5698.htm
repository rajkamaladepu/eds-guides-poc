Search HTTP Service Requires SSL (Computer Properties Indexing setting)

Description
Specifies whether or not the Index Query Server HTTP service requires SSL (HTTPS) connections. If this setting is on, a certificate must be bound to the appropriate port on the Index Query Server HTTP service host.
Note:
Do not change this advanced setting unless your technical support provider advises you to do so.
Supported values
  • Off (default).
  • On.
