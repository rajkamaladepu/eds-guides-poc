Search Logs Folder (Computer Properties Indexing setting)

Search Logs Folder (Computer Properties Indexing setting)

Description
A local folder to store log files that are generated by the Index Query Server.
Note:
Do not change this advanced setting unless your technical support provider advises you to do so.
These logs are only written if the Search Log Queries setting or the Search Log Results setting is turned on.
See Search Log Queries (Computer Properties Indexing setting).
See Search Log Results (Computer Properties Indexing setting).
Supported values
  • A text value specifying a local folder. The default is that no folder is specified and the Vault Service account TEMP folder is used.
