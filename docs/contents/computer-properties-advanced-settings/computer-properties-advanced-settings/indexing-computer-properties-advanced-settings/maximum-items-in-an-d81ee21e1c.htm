Maximum items in an internet mail index volume (Computer Properties Indexing setting)

Description
The maximum number of items in an internet mail index volume. The index volume rolls over to a new volume when it contains approximately this number of items.
Note:
Do not change this advanced setting unless your technical support provider advises you to do so.
Supported values
  • An integer specifying a number of items. The default is 5000000.
