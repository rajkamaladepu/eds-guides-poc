Search HTTP Service Path (Computer Properties Indexing setting)

Description
The path part of the URL for the Index Query Server HTTP service, as used by IndexClient.
Note:
Do not change this advanced setting unless your technical support provider advises you to do so.
Supported values
  • A text value specifying a path. The default is:
/enterprisevault/search/indexserversearchservice/
