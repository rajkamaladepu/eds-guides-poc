Search Log Queries (Computer Properties Indexing setting)

Description
Specifies whether or not the Index Query Server logs queries to disk. The Search Logs Folder setting specifies the location for the files.
See Search Logs Folder (Computer Properties Indexing setting).
Note:
Do not change this advanced setting unless your technical support provider advises you to do so.
Supported values
  • Off (default).
  • On.
