Maximum concurrent indexing capacity (Computer Properties Indexing setting)

Description
The maximum total of index volumes and subtasks that can be processed concurrently.
Note:
Do not change this advanced setting unless your technical support provider advises you to do so.
Supported values
  • An integer specifying a number of index volumes and subtasks. The default is 30.
