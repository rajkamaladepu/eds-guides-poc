Maximum child process shutdown time (Computer Properties Indexing setting)

Description
The maximum time the Indexing service waits for each child process to shut down, in minutes.
Note:
Do not change this advanced setting unless your technical support provider advises you to do so.
Supported values
  • An integer specifying a number of minutes. The default is 15.
