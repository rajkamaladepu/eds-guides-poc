Indexing Max Request Length (Computer Properties Indexing setting)

Description
The indexing HTTP runtime maximum request length, in megabytes.
Note:
Do not change this advanced setting unless your technical support provider advises you to do so.
Supported values
  • An integer specifying a number of megabytes. The default is 100.
The minimum supported value is 1 MB and the maximum supported value is 500 MB.
