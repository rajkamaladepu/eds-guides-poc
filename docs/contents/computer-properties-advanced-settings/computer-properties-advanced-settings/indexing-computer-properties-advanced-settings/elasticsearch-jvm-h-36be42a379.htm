Elasticsearch JVM heap size (MB) (Computer Properties Indexing setting)

Description
Specifies the Elasticsearch JVM heap size in megabytes (MB) that you want to allocate and reserve for the Indexing engine. The minimum value is 1024 MB, and the maximum is 50% of memory, with a maximum limit at 30720 MB.
Supported values
An integer specifying the number of megabytes. The default is 8192.
