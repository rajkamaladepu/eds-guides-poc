Elasticsearch ports range (Computer Properties Indexing setting)

Description
The Elasticsearch Indexing engine uses the first two open ports from this range. If required, you can specify a new range from the supported range (1-65535). For example, 11000-12000.
Supported values
Range of 2 integers. The default is 9200-9300.
