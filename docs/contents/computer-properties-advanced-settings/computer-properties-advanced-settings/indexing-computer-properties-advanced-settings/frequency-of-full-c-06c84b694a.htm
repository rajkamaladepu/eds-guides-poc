Frequency of full checks for index volumes to process (Computer Properties Indexing setting)

Description
The time between the Indexing service's full checks for index volumes to process, in hours.
Note:
Do not change this advanced setting unless your technical support provider advises you to do so.
Supported values
  • An integer specifying a number of hours. The default is 10.
