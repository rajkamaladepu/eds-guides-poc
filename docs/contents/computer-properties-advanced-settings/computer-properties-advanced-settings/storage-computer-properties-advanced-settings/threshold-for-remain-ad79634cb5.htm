Threshold for remaining disk space (Computer Properties Storage setting)

Description
The minimum percentage of disk space that must be available at the storage queue location. If is limit is reached, Enterprise Vault does not place any new items on the queue until more space is available.
If you keep safety copies on the storage queue and this limit is reached, the cause may be that there has been no recent backup of the vault store.
Supported values
  • An integer specifying the minimum percentage of free disk space. The default is 10.
