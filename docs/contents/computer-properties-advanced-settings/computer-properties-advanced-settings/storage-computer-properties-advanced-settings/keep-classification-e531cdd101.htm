Keep classification files (Computer Properties Storage setting)

Keep classification files (Computer Properties Storage setting)

Description
Controls whether to keep the temporary files that the Enterprise Vault classification component makes when it classifies items. These files may contain sensitive data, so it is normally a good idea to delete them. On the other hand, if Enterprise Vault does not classify items as you expect, you may be able to find out why by examining the files.
Supported values
  • Off (default).
  • On.
