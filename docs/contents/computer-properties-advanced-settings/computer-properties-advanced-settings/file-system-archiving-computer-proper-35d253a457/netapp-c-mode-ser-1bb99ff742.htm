NetApp C-Mode server port number (Computer properties File System Archiving setting)

Description
Lets you specify the port number that the ONTAPI Management API should use to communicate with the NetApp C-Mode server. The port number should be same as the one that is configured on the NetApp C-Mode server.
Supported values
An integer specifying a port number. Typically, port 443 is used for HTTPS connections and port 80 is used for HTTP connections.
