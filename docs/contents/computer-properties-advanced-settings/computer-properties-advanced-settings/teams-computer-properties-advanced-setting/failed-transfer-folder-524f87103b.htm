Failed transfer folder processing interval (Computer Properties Teams setting)

Description
Interval in minutes for processing failedTransfer folder of a Teams Importer when Merge1 webhook is called.
Note:
Do not change this advanced setting unless your technical support provider advises you to do so.
Supported values
  • An integer specifying a number of minutes. The default is 30.
  • Minimum value can be 1.
  • Maximum value can be 60.
