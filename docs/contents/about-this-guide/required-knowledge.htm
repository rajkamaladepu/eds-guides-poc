Required knowledge

To administer Enterprise Vault, you need a working knowledge of the following:
  • Windows Server administrative tasks
  • Microsoft SQL Server
  • Microsoft Message Queue Server
  • Internet Information Services (IIS)
  • Your archive storage hardware and software
If you intend to use Enterprise Vault with Domino server, Microsoft Exchange, or Microsoft SharePoint, you also require a working knowledge of these products.
