Valid Enterprise Vault site aliases (Exchange Archiving General setting)

Description
A semicolon-separated list of the Enterprise Vault site aliases that are currently in operation in your environment.
During shortcut processing, Enterprise Vault does not attempt to make connections to any site that is not in this list. This prevents connection attempts to Enterprise Vault sites that no longer exist in your environment.
Supported values
  • A semicolon-separated list of Enterprise Vault site aliases. You can enter unqualified aliases or fully qualified aliases, but they must match the site aliases you specified when you first ran the Enterprise Vault Configuration wizard. If you created unqualified site aliases in the Configuration wizard, you must enter unqualified aliases here.
  • If this string is empty, Enterprise Vault attempts to process all shortcuts.
