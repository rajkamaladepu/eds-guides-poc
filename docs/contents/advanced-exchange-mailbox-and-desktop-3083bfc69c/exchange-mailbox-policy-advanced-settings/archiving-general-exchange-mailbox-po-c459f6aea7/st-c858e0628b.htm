Strip attachments to non-shortcut items (Exchange Archiving General setting)

Description
Controls whether attachments are removed from calendar items, meeting items (including requests, responses and cancellations), task and task request items, and contacts after archiving. These items are not changed into Enterprise Vault shortcuts when they are archived. If Enterprise Vault removes an attachment, it replaces it with a link to the attachment.
Supported values
  • On (default). Attachments are removed from non-shortcut items after archiving.
  • Off. Attachments are not removed from non-shortcut items after archiving.
