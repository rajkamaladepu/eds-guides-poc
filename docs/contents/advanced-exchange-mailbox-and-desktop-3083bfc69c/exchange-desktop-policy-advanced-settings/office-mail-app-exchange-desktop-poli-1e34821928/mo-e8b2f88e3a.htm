Mode (Exchange Office Mail App setting)

Mode (Exchange Office Mail App setting)

Description
Determines whether Outlook 2013 and OWA 2013 users have access to the full Enterprise Vault functionality.
Supported values
  • Full. Users can choose a retention category for each item that they manually archive.
  • Light (default). Users cannot choose a retention category for each item that they manually archive. Instead, Enterprise Vault archives the item with the default retention category for the mailbox folder that contains the item.
