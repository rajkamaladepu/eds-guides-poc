Max item updates per synchronization (Exchange Virtual Vault setting)

Description
Controls the maximum number of property change requests during a Vault Cache synchronization. Any remaining requests are made at the next synchronization.
Update requests use relatively few resources on the Enterprise Vault server.
Supported values
  • An integer value. The default is 0 (no limit).
