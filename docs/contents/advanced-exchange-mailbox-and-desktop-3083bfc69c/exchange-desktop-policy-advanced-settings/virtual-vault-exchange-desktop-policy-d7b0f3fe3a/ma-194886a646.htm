Max delete requests per synchronization (Exchange Virtual Vault setting)

Description
Controls the maximum number of delete requests during a Vault Cache synchronization. Any remaining requests are made at the next synchronization.
Deletion requests use relatively few resources on the Enterprise Vault server.
Supported values
  • An integer value. The default is 0 (no limit).
