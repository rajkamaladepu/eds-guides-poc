Max total size of contentless operations (Exchange Virtual Vault setting)

Description
Controls the maximum total size in megabytes of copy and move operations when items have no content in Vault Cache. This setting does not apply to documents that are placed directly in the mailbox. It only applies to standard Outlook mail types, for example, mail items, calendar items, tasks, and contacts.
This setting only applies when two or more items with no content are involved in the operation. Retrieval of one item is allowed regardless of its size.
Supported values
  • An integer value. The default is 64 (MB). The value 0 specifies no limit.
