Public Folder operations (Exchange Outlook setting)

Description
Controls whether users can do the following:
  • Manually archive from public folders.
  • Manually restore from public folders.
  • In public folders, delete shortcuts and their corresponding archived items.
Supported values
  • Off (default). Users can archive, restore, or delete in public folders. Users must be enabled for mailbox archiving and must have Editor, Publishing Editor, or Owner permissions on the folders that they want to modify.
  • On. Users cannot archive, restore, and delete in public folders.
Legacy name
DisablePublicFolderOps
