PST search interval (Exchange Outlook setting)

Description
When using client-driven PST migration, this controls the amount of time that the Outlook Add-In waits after searching a user's computer for PST files before searching again.
Supported values
  • Integer. Indicates the number of days to wait between searches.
The default is 7 (days).
