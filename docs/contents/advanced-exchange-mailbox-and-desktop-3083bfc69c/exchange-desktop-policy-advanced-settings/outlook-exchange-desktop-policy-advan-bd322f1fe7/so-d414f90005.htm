Soft deletes (Exchange Outlook setting)

Description
Controls the behavior when a user deletes a shortcut and an archived item. The default is to perform a permanent deletion of the shortcut; that is, a deletion that is not recoverable in Exchange. If the permanent deletion fails, a recoverable deletion is performed.
If recoverable deletion is unacceptable, set the value to Off.
Supported values
  • On (default). Allow recoverable deletion.
  • Off. Do not allow recoverable deletion and display an error message instead.
Legacy name
