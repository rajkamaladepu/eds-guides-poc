Force form reload on error (Exchange Outlook setting)

Description
Controls whether the Outlook registry entry ForceFormReload is written to force Outlook to reload forms on error.
Note that this registry entry applies to all Outlook forms, so changing this setting may affect other applications.
Supported values
  • Remove entry. Remove the registry entry. This means that Outlook does not reload forms on error.
  • Write entry (default). Write the registry entry. This forces Outlook to reload forms on error.
Legacy name
SetForceFormReload
