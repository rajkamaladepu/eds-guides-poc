Allow script in shared folders (Exchange Outlook setting)

Description
Controls whether the Outlook setting Allow Script In Shared Folders is automatically set by the Outlook Add-In. This setting is required in order for the Outlook Add-In to be able to open shortcuts that are in shared folders.
Supported values
  • Force off. The Outlook Add-In cannot open shortcuts that are in shared folders.
  • Force on (default). The Outlook Add-In can open shortcuts that are in shared folders.
  • Keep user's setting. The user's Outlook setting of Allow Script In shared Folders is not changed.
