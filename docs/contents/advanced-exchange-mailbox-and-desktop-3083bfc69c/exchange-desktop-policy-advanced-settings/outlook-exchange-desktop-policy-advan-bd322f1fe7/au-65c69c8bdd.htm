Automatically re-enable Outlook add-in (Exchange Outlook setting)

Description
Controls whether the Enterprise Vault Outlook Add-In is automatically re-enabled if it has been disabled.
Supported values
  • Do not re-enable. The Outlook Add-In is never automatically re-enabled if it has been disabled.
  • Re-enable and tell user (default). The Outlook Add-In is automatically re-enabled if it has been disabled and a message informs the user that this has happened.
  • Re-enable silently. The Outlook Add-In is automatically re-enabled if it has been disabled and the user is not informed.
  • Ask user. The user is asked whether the Outlook Add-In should be re-enabled.
