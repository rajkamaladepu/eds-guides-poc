Shortcut download progress (Exchange Outlook setting)

Description
Controls the display of the progress dialog box that appears when a user opens a shortcut and the item is being downloaded to the user's computer.
Supported values
  • 0\. Do not display progress dialog box at all.
  • An integer greater than zero. Show the progress dialog box after this number of seconds.
Default is 1 (second).
Legacy name
