Automatically delete IE file cache (Exchange Outlook setting)

Description
Controls whether the Enterprise Vault add-in to Outlook automatically deletes a user's Temporary Internet files if the Internet Explorer cache is preventing archived items from being displayed.
Supported values
  • Do not delete files (default). Temporary Internet files are never deleted.
  • Delete files and tell user. The Temporary Internet files are deleted and a message informs the user that they have been deleted.
  • Delete files silently. The Temporary Internet files are deleted and the user is not informed.
  • Ask user. The user is asked whether the Temporary Internet files should be deleted.
