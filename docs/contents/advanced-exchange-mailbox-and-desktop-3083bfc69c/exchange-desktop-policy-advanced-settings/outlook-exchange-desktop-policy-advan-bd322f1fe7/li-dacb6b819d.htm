Limit automatic re-enabling of add-in (Exchange Outlook setting)

Description
Controls the maximum number of times in any seven-day period that the Outlook Add-In can re-enable itself as an Outlook add-in.
Supported values
  • An integer specify the maximum number of times that the Outlook Add-In can re-enable itself in any seven-day period. The default is 3.
