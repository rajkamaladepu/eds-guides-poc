PST Import work check interval (Exchange Outlook setting)

PST Import work check interval (Exchange Outlook setting)

Description
The number of minutes client-driven PST migration waits, having completed its work, before checking for more work.
Supported values
  • Integer. An integer value specifying the number of minutes to wait before checking for more work.
The default is 60.
