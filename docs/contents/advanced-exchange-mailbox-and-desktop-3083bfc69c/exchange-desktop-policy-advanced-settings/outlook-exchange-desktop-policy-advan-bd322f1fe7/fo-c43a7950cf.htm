Forward original item (Exchange Outlook setting)

Description
Controls the behavior when a user forwards a shortcut. The default is to forward the archived item, but it is possible to forward the contents of the shortcut itself.
Supported values
  • Shortcut. Forward the shortcut.
  • Original (default). Forward the archived item.
Items of type IPM.Document or IPM.Appointment cannot be forwarded. If a user tries to forward one of these an explanatory message is displayed.
