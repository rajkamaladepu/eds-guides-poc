Event filtering examples

Some event filtering examples:
  • To generate an event filtering summary event every 10 minutes instead of the default of 15 minutes, set ReportConfigPeriodMinutes to 10.
  • To specify that events 1234, 5678, and 4133 must never be suppressed, set NeverSuppressEventIDs to the following:
    1234;5678;4133
