Importing the Enterprise Vault SCOM management pack

The SCOM management pack is provided in a self-extracting executable file, Veritas Enterprise Vault Management Pack.exe. The file is on the Enterprise Vault media.
To import the Enterprise Vault management pack
  1. Load the Enterprise Vault media.
  2. If Windows AutoPlay is enabled on the server, Windows shows an AutoPlay dialog box. Click Run Setup.exe.
If AutoPlay is not enabled, use Windows Explorer to open the root folder of the installation media and then double-click the file Setup.exe.
  1. In the list in the left pane of the Veritas Enterprise Vault Install Launcher window, click Enterprise Vault.
  2. Click Client Installation.
  3. In the right pane, click SCOM Management Pack.
  4. Click Open Folder. Windows Explorer starts in the folder that contains the management pack.
  5. Double-click the following file:
Veritas Enterprise Vault Management Pack.exe
  1. Read the displayed license and click Yes to agree to the terms.
  2. Specify the folder where you want the place the management pack and click OK.
  3. In System Center Operations Manager select Administration.
  4. In the left pane, right-click Administration and click Import Management Packs.
  5. In the Import Management Packs Wizard, click Add and then click Add from disk.
  6. If there is a prompt that asks whether you want to search the online catalog for dependencies, click No.
  7. Navigate to the folder that contains the management packs that you extracted.
  8. Click Veritas.EnterpriseVault.Library.mp and Veritas.EnterpriseVault.12.mp and then click Open. The wizard adds the management packs to the Import list.
  9. Click Install. The wizard imports the management packs.
  10. Click Close.
