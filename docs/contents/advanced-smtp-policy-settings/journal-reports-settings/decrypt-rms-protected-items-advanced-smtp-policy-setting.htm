Decrypt RMS Protected Items (Advanced SMTP policy setting)

Description
If journal report decryption is enabled on Exchange, then two messages are attached to the journal report during archiving: the original RMS-protected message and a clear text version. If journal report decryption is disabled, then Enterprise Vault stores only the RMS-protected message along with the journal report during archiving. This policy setting controls whether Enterprise Vault should decrypt the RMS-protected messages for indexing and preview.
Supported values
  • Do not decrypt (default)
  • Decrypt for journal archives only
