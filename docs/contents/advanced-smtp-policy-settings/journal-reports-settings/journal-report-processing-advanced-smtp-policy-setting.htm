Journal report processing (Advanced SMTP policy setting)

Description
Controls whether journal reports are processed and stored with the message.
Supported values
  • Process journal reports for journal archives only (default). Journal reports are discarded for user archive types, such as Exchange Mailbox and Internet Mail archives. Only the original messages are processed and archived.
For SMTP, Exchange Journal, Domino Journal, and Shared archives, journal reports are processed and archived with the original message.
  • Process journal reports for all archives. For all archive types, journal reports are processed and archived with the original messages. As this option exposes BCC recipient details, you should only select it for archives that are used for compliance and discovery purposes.
  • Discard journal reports for all archives. For all archive types, journal reports are discarded and only the original messages are processed and archived. Select this option if users have access to archives that contain journaled SMTP messages.
