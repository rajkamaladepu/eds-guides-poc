Roles and Enterprise Vault Operations Manager

Any user other than the Vault Service account must be assigned to a suitable administrator role to access the Enterprise Vault Operations Manager web application. Users can view only the tabs and tables in Operations Manager that are applicable to the role to which they are assigned. The tabs and tables that are available for a role are consistent with the containers that are available to that role in the Administration Console. The Power Administrator role is able to see all the tabs and tables in Operations Manager.
