Working with predefined RBA roles

Enterprise Vault provides four PowerShell cmdlets that let you manage RBA roles and role membership. You must log in using the Vault Service account to use all these cmdlets.
Table: Cmdlets for managing role membership
Cmdlet
Description
Get-EVRBARole
This cmdlet lists the RBA roles that exist in the Enterprise Vault Directory.
Get-EVRBARoleMember
This cmdlet lists the members of an RBA role.
Add-EVRBARoleMember
This cmdlet adds members to an RBA role.
Remove-EVRBARoleMember
This cmdlet removes members from an RBA role.
