Preemptive archiving threshold (Domino Vault Cache setting)

Description
Specifies the number of days before archiving at which Enterprise Vault copies items preemptively to the Vault Cache, in anticipation that the items will soon be archived. The copy takes place on the user's computer, with no further download required.
Supported values
  • Integer value, specifying a number of days. The default is 7.
