Strip attachments to non-shortcut items (Domino Archiving General setting)

Description
Controls whether attachments are removed from appointments and tasks after archiving. These items are not changed into Enterprise Vault shortcuts when they are archived. If Enterprise Vault removes an attachment, it replaces it with a link to the attachment.
Supported values
  • On (default). Attachments are removed from non-shortcut items after archiving.
  • Off. Attachments are not removed from non-shortcut items after archiving.
Legacy name
StripAttachmentsToNonShortcutItems
