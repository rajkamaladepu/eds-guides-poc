Reset archive names (Domino Archiving General setting)

Description
Controls whether archive names are kept synchronized with Domino user names. If the setting is On, any changes to the Domino user name are applied to the archive name when the Domino Provisioning task runs, either on a scheduled run or when the administrator clicks Synchronize Individual Mailboxes on the task properties.
Supported values
  • On (default). Archive names are updated during synchronization.
  • Off. Archive names are not updated during synchronization.
